# Голосовой ассистент с локальной LLM

Этот проект реализует голосового ассистента на Python, похожего на Siri, который работает локально и использует LLM (через Ollama) для ответов на вопросы.

## Особенности
- **Распознавание речи**: Использует `speech_recognition` с `sounddevice` (вместо `PyAudio`, что решает проблемы установки на Windows/Python 3.14+).
- **Синтез речи**: Использует `pyttsx3` (работает офлайн). Если синтез недоступен, ответы выводятся текстом.
- **Интеллект**: Использует локальную нейросеть через Ollama.

## Требования

1.  **Python 3.8+** (протестировано на 3.14).
2.  **Ollama**: Для работы локальной нейросети.
3.  **Микрофон и динамики**.

## Установка и настройка

### 1. Установка Ollama и модели

1.  Скачайте и установите [Ollama](https://ollama.com/) для вашей ОС.
2.  После установки откройте терминал (PowerShell или CMD) и скачайте модель:
    ```bash
    ollama pull llama3
    ```
    *Примечание: Вы также можете использовать другие модели, например `mistral` или `gemma`, изменив переменную `MODEL_NAME` в файле `main.py`.*

### 2. Установка зависимостей Python

В папке проекта выполните команду:
```bash
pip install -r requirements.txt
```

### 3. Запуск ассистента

Запустите файл `main.py`:
```bash
python main.py
```

## Использование

1.  Когда ассистент запущен, он будет слушать микрофон.
2.  Скажите **"Сервер привет"** (или "Server privet"), чтобы активировать его.
3.  После ответа "Слушаю вас", задайте свой вопрос.
4.  Ассистент подумает (отправит запрос в локальную LLM) и озвучит ответ.

## Настройки

В файле `main.py` можно изменить:
-   `WAKE_WORD`: Фраза для активации (по умолчанию "сервер привет").
-   `MODEL_NAME`: Имя модели Ollama (по умолчанию "llama3").
-   `OLLAMA_URL`: Адрес API Ollama (по умолчанию http://localhost:11434/api/generate).

## Возможные проблемы
- **Ошибка TTS (синтеза речи)**: Если вы видите ошибку "Класс не зарегистрирован", ассистент будет работать в текстовом режиме (выводить ответы в консоль). Это может быть связано с версией Python или настройками Windows.
